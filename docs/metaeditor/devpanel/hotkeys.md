*Hotkeys are available in production, don't forget to disable hotkeys if needed.*

| Press    | Description                        |
| -------- | ---------------------------------- |
| `Ctrl+Z` | Show debug panel                   |
| `Ctrl+R` | Manual restart cloud stream server |
| `Ctrl+F` | Fullscreen mode                    |
